# 📊 Proyecto BBDD SQL – FullStack - Data Science

## 📌 Descripción

Este proyecto consiste en el diseño, implementación y explotación de una base de datos relacional utilizando SQL, como parte del módulo de Bases de Datos compartido entre los programas de Data Science y FullStack en la escuela The Bridge en la promoción de Marzo 2026.

El objetivo principal es aplicar los conocimientos adquiridos en modelado de datos, normalización, consultas SQL y análisis de información para resolver un problema real a partir de un dataset.

---

## 🎯 Objetivos

- Diseñar un modelo entidad-relación (E/R)
- Transformar el modelo a esquema relacional
- Crear la base de datos en SQL
- Poblar las tablas con datos
- Validar la base de datos mediante queries de testeo

## 🧪 Validación de la BBDD (queries.sql)

El archivo `queries.sql` incluye un conjunto de consultas diseñadas para verificar el correcto funcionamiento de la base de datos y facilitar su exploración.

Estas queries permiten asegurar que:

- Las tablas están correctamente pobladas  
- Las relaciones funcionan como se espera  
- Los datos son coherentes

## 🧠 Modelo de Datos

El diseño de la base de datos se ha realizado siguiendo buenas prácticas:

- Normalización (hasta 3FN)
- Uso de claves primarias y foráneas
- Integridad referencial
- Tipos de datos adecuados
